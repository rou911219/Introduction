# 👋 Introduction

您好，我目前就讀於輔仁大學統計資訊學系應用統計碩士班，專長為 📊 統計分析、📈 資料處理與 🎨 海報設計。
熟悉 R、Python 等分析工具，研究領域包含迴歸分析、測量誤差模型與資料視覺化，並致力於將複雜的數據內容，以清楚且具美感的方式呈現。
除了數據分析之外，我也喜歡透過海報、簡報與視覺設計提升資訊傳達效果，希望結合邏輯分析與創意設計，讓資料不只是數字，而是更容易被理解的故事。
✨ 此 GitHub 主要用於分享我的研究、程式碼、資料分析與設計相關作品。
